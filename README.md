# java-web-maven-spring-jsp-mysql

## Description
A POC for springboot web app with jsp support
connecting to a database.

## Tech stack
- java
- maven
  - springboot
  - jsp
  - mysql connector
- bootstrap
- jquery
- dataTable

## Docker stack
- mariadb
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
